# Game Arena

A replicated version of the Game Arena website from https://game-arena-theta.vercel.app/

## Features

- Responsive design
- Interactive gaming website
- Local server setup

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open your browser and go to http://localhost:8081

## Structure

- `index.html` - Main HTML file
- `style.css` - Custom styles
- `script.js` - JavaScript functionality
- `assets/` - Images and other assets
- `favicon.png` - Favicon
